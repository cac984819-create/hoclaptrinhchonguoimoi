Lộ Trình Học CSS Chuyên Sâu

Phần 1: CSS Box Model & Selectors

  • Box Model: Hiểu rõ cách tính kích thước dựa trên content, padding, border, và margin.
  • Selectors: Từ cơ bản (Class, ID) đến nâng cao (Pseudo-classes như :hover, :nth-child).

Phần 2: Modern Layout (Flexbox & Grid)

  • Flexbox: Điều hướng bố cục 1 chiều (hàng hoặc cột).
  • Grid: Xây dựng hệ thống lưới 2 chiều phức tạp cho toàn bộ trang web.
  • Responsive Design: Sử dụng @media queries để web tự tương thích điện thoại.
