Advanced CSS3 Pipeline

Từ quy trình Render của trình duyệt đến các hệ thống bố cục phức tạp.

1. CSSOM & Quá trình Render

Trình duyệt không chỉ dùng HTML. Nó kết hợp DOM và CSSOM (CSS Object Model) để tạo ra Render Tree.

  • Recalculate Style: Tính toán xem selector nào khớp với phần tử nào.
  • Layout (Reflow): Tính toán kích thước và vị trí hình học của từng "hộp".
  • Paint: Điền màu, hình ảnh, biên giới vào các pixel.
  • Composite: Xếp chồng các lớp (layers) lên nhau để hiển thị.

2. Box Model & Stacking Context

Để trở thành chuyên gia, bạn phải hiểu cách các "hộp" tương tác trong không gian 3D của trình duyệt.

Box-sizing: border-box

Luôn sử dụng thuộc tính này để width bao gồm cả padding và border, giúp kiểm soát kích thước chính xác 100%.

Kỹ thuật Stacking: z-index chỉ hoạt động khi bạn có position (relative, absolute, fixed). Nó tạo ra một "Stacking Context" mới.

3. Layout Mastery: Flexbox & Grid

CSS Grid (Hệ thống lưới 2 chiều)

Grid mạnh nhất khi xây dựng bố cục lớn. Khái niệm fr (fractional unit) giúp chia tỉ lệ màn hình cực kỳ linh hoạt.

.container { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; }

Flexbox (Hệ thống 1 chiều)

Dùng cho các thành phần con trong một khối. Hãy nắm vững flex-grow, flex-shrink và flex-basis để xử lý co giãn giao diện mượt mà.

4. Scalable CSS: Biến & Quy tắc BEM

Trong các dự án lớn, việc quản lý mã CSS rất khó khăn. BEM (Block Element Modifier) là cứu cánh.

/* Ví dụ về BEM */ .card { } /* Block */ .card__title { } /* Element */ .card--featured { } /* Modifier */ /* Sử dụng CSS Variables */ :root { --brand-color: #2965f1; } .button { color: var(--brand-color); }
